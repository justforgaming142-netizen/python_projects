This is meant to be a simple command line budget-tracking utility that enables the user to  create and manage data entries on a json file using python.
This is a work in progress.
The main features it intends to have are:
    make and manage dict that is transformed to a .json file
    summaries the total income and expanses
    summaries the above on a monthly basis
    category wise summary
    backing up(little unlikly but I'll try)
